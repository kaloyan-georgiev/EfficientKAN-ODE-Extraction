# Symbolic Extraction of Nonlinear Differential Relationships

This repository contains the codebase for extracting symbolic nonlinear differential equations using Kolmogorov-Arnold Networks (KANs). We utilize the Schättler tumor-immune-angiogenesis ODE framework as a highly complex, non-polynomial mathematical testbed.

## Project Structure

* **`dataset_generation.py`**: Contains the logic to synthesize the dataset using a discrete-time forward Euler mapping of the Schättler ODEs.
* **`models.py`**: Houses the architecture definitions for both the `efficient-kan` implementation (B-Spline KAN) and the baseline Multi-Layer Perceptron (MLP). Both models are strictly constrained to $\sim$50,150 parameters to ensure an iso-parameter comparison.
* **`train.py`**: The main training loop. It automatically generates the dataset, trains both the MLP and KAN models side-by-side, outputs performance benchmarks, and saves the trained KAN model to disk.
* **`run_symbolic_regression.py`**: The symbolic extraction pipeline. It loads the trained KAN, identifies unpruned active edges using $L_1$ thresholds, fits 1D analytical functions to these edges using `PySR`, and reconstructs the macroscopic ODEs via `SymPy`.
* **`biological_dataset.csv:`** Contains a sample dataset for inspection purposes

## Requirements

Ensure you have the following installed:

* `torch`
* `numpy`
* `sympy`
* `pysr` (Requires Julia backend that will be installed automatically when the file is ran)

## Usage

### Phase I: Training

To begin, simply run the training script:

```bash
python train.py
```

**What this does:**

1. Automatically invokes `dataset_generation.py` to synthesize 10,000 biological state transitions.
2. Initializes both the KAN and the MLP to verify they have identical parameter capacity.
3. Trains both networks over the specified number of epochs using an AdamW optimizer and $L_1$ weight decay on the KAN grid coefficients.
4. Outputs the final Training MSE, Evaluation MSE, and Evaluation MAE.
5. Saves the trained KAN weights to `kan_model.pth`.

### Phase II: Symbolic Extraction

Once `kan_model.pth` has been successfully generated by the training script, run the symbolic regression pipeline:

```bash
python run_symbolic_regression.py
```

**What this does:**

1. Loads the trained KAN model.
2. Scans the network layers to isolate edges with an $L_1$ norm exceeding the pruning threshold $\epsilon=10^{-3}$.
3. Passes 1D uniform points through the surviving edges to generate a localized dataset.
4. Uses `PySR` to find the optimal symbolic expressions (e.g., $x^2$, $\sin(x)$, $\exp(x)$) for each isolated edge.
5. Leverages `SymPy` to algebraically sum the discovered 1D equations based on the network's macroscopic additive topology, reconstructing and printing the final multivariate ODEs.
